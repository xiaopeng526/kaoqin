# 考勤管理-前端项目


### 结构说明
* attd 考勤管理
* brief 简历管理
* contract 合同管理
* customer 客户管理
* leave 请假管理
* mywork 我的工作/工作列表
* outgoing 外出管理
* overtime 加班管理
* project 项目管理
* remind 提醒列表
* states 统计分析
* team 团队管理
* timeline 时间线
* worklog 日报周报
* role 角色管理
* user 用户管理
* dept 部门管理
